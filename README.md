# 🐍 Python Foundations for Data Science  
### A Structured Learning Portfolio

---

## 📌 Overview
This repository documents my progression through **core Python programming concepts**, with a focus on building a strong foundation for **data science, analytics, and applied research**.

The project is organized as a series of Jupyter notebooks, each covering a key concept and applying it through practical exercises and mini-projects.

---

## 🎯 Objectives
- Develop fluency in Python fundamentals  
- Build problem-solving skills through structured exercises  
- Prepare for data science workflows using Python  
- Transition from programming basics to applied data analysis  


---

## 🧠 Topics Covered

### 🔹 Python Fundamentals
- Variables and data types  
- Strings and basic operations  
- Input/output  

### 🔹 Control Flow
- Conditional statements (`if`, `else`, `elif`)  
- Logical operators  

### 🔹 Data Structures
- Lists  
- Dictionaries  

### 🔹 Iteration
- `for` loops  
- `while` loops  

### 🔹 Functions
- Function definition and usage  
- Functions with return values
